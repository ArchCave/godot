# O. Jooyoung — Godot Game Development Repository

This repository is the official Godot engine project for author O. Jooyoung. It collects prototypes, scenes, scripts, assets, and documentation created during the iterative development of small to medium-sized Godot projects. The repository is organized to facilitate reproducible builds, straightforward collaboration, and efficient iteration.

## Purpose
- Rapidly prototype gameplay ideas and validate core mechanics.
- Maintain a clear, consistent folder structure for scenes, resources, and assets.
- Encourage disciplined version control and review practices to improve code quality and maintainability.

## Repository Layout (high level)
- `project.godot` — Godot project configuration file
- `main.tscn` — example entry scene
- `Sprites/` — sprite sheets and image assets
- `Materials/` — materials, lighting examples, and related scenes

## Requirements & Running the Project
1. Install Godot Engine. Confirm the recommended engine version in `project.godot` if necessary.
2. Clone the repository:

```bash
git clone <repository-url>
```

3. Open Godot, select the `project.godot` file, then run a scene such as `main.tscn` from the editor to test the project.

## Development Practices
- Follow established GDScript conventions and Godot community best practices for naming and structure.
- Keep commits focused and message them clearly when adding assets or changing scenes.
- Implement features and fixes in small, reviewable increments (prefer pull requests for collaborative work).
